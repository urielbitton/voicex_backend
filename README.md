# Voicex Backend

This backend server is built with Fastify and MongoDB. 

*This repo uses ***yarn*** as the package manager.*

<br>
<br>

## Install dependencies

After cloning the repo, navigate into the folder. In the terminal, run
<br>
`yarn install`
<br>
<br>

## Start the server

In the terminal, run
<br>
`yarn start`
<br>
<br>
